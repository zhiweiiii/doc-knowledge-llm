
项目介绍：
这是一个基于Qwen模型的文档知识问答系统

环境：
请先安装docker环境
由于模型文件较大，所以请自行下载model.safetensors模型文件

地址：
https://www.modelscope.cn/models/Qwen/Qwen3-0.6B/resolve/master/model.safetensors

下载后把model.safetensors放到model目录下即可

启动命令：
````
./build.sh
````
启动后访问：
http://localhost:81
